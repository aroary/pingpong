# pingpong
#### 2D Ping Pong game
![preview image](https://cdn.discordapp.com/attachments/806009448597946393/899472639352000512/unknown.png)
___
### How to play:
* player 1
    * `w` To move **up**
    * `s` To move **Down**
* player 2
    * `up` To move **up**
    * `down` To move **Down**
* Press `Space` to **serve**
___
#### To change the game settings, you can change the values in the `./settings.json` file.
