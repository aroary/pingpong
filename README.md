# pingpong
#### 2D Ping Pong game
___
### How to play:
* player 1
    * `w` To move **up**
    * `s` To move **Down**
* player 2
    * `up` To move **up**
    * `down` To move **Down**
___
#### To change the game settings, you can change the values in the `./settings.json` file.
